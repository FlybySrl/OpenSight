Detailed Description

Base class that defines the common functions and the interface to manage the modules in an application. Inherits from QMainWindows so that it can be used also as the main widget of a graphical application.

virtual void fby::AppBase::ConfigModules ( )

Configures the modules managed by this AppBase. This is a pure virtual function that needs to be implemented in subclasses in order to properly set the managed modules.

Note
In order to add a module to be manages, the user must call the method AddModule().
