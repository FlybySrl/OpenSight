virtual QToolBar* fby::ModuleWrapperGUI::CreateToolBar ( )
virtual

CreateToolBar creates a QToolBar with default actions. The default actions are the following:

  • Start
  • Stop
  • Pause
  • <separator>
  • Clear
  • Hide

If a toolbar for this ModuleWrapperGUI already exists, then it will NOT be created.

Returns
the new QToolBar or the existing one if it has been previously created.
